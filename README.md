# 80-3-topshiriq
Shifoxonada navbat olish smart-kontraktlarini tuzing. Kontraktga payable funksiyani qo‘shing. Faqat ma’lum adresdan kelgan to‘lovni qabul qilish shartini yozing. Minimal to‘lov miqdorini (require) bilan tekshiring. Agar shart bajarilsa, mablag‘ni boshqa adresga o‘tkazing. if/else yordamida turli holatlarda turli tranzaksiyalarni amalga osh
